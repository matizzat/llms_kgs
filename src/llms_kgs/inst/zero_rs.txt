Your task is to return a list of semantic relationships we will later use to build Knowledge Triples. Some examples are: "lives_in", "occurred_in", "is_a". The output should be in the following format:

semantic_relation_phrase
semantic_relation_phrase
...
semantic_relation_phrase

Example output:
